Vole-Machine simulation

The Vole Machine Simulator is a C++ program that emulates a simple virtual machine capable of interpreting custom machine language instructions. It uses two main components: Registers: 16 registers (keys '0' to 'F'), each initialized to "0", for temporary data storage. Memory: 256 memory locations, addressed by hex string keys from "00" to "FF", also initialized to "0". Instructions are read from a file, stored in a vector, and processed in sequence. For example, 2018 adds decimal 18 to register 0, while 35B1 updates a specific memory location. This simulator provides a foundation for learning machine language and memory management concepts.
